## Constraints
1. **資料來源限制**  
   - 只能從 **GitHub Issue / Comment** 抽取事實；其他來源（如聊天記錄、非結構化筆記）不列入正式記憶。  
2. **手動筆記保護**  
   - `shared/manual.md` 為唯一的手動長期記憶檔案，**不可被自動流程覆寫**。  
3. **摘要原則**  
   - 任何新增的長期記憶必須 **蒸餾** 成可重用的規則或決策，避免逐段複製原始 Issue 內容。
